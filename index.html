OSM QA tiles

OSM QA tiles are a vector tile set created to be used for data analysis of OpenStreetMap with Tile Reduce.

Unlike vector tiles used for displaying stylized maps, OSM QA tiles include:

  • unsimplified geometries
  • all OpenStreetMap tags
  • additional properties like the changeset, time, and the user that edited a feature

Mapbox uses these vector tiles to do analysis of OpenStreetMap; conflating with other large datasets like TIGER from the US Census, checking for broken features, detecting connectivity issues in the road network, and detecting vandalism.
